# template.front

Шаблон для создания интерфейсов Vue + TS + CSS.

**Node:** 20 LTS

**Vue:** 3

## Начало работы

* Установите [pmpm](https://pnpm.io/installation)
* `pnpm install`
* `pnpm dev`

## Работа с проектом

### API

Все роуты должны быть прописаны в [api.d.ts](src%2Ftypes%2Fapi.d.ts)
в виде дискриминантных связей.
